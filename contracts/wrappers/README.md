# Contract Wrappers

This directory will contain TypeScript wrapper classes generated by the Blueprint
build tool. Run the following command to compile the Tact contracts and generate
the wrappers:

```bash
# Install Blueprint (once)
npm install -D @ton/blueprint @ton/sandbox @ton/test-utils

# Compile contracts and generate wrappers
npx blueprint build
```

After a successful build, this directory will contain:

| File | Description |
|------|-------------|
| `AgentWallet.ts` | Wrapper for `contracts/agent-wallet.tact` |
| `AgentFactory.ts` | Wrapper for `contracts/agent-factory.tact` |
| `StrategyExecutor.ts` | Wrapper for `contracts/strategy-executor.tact` |

The wrappers expose type-safe `send()` and `get()` methods matching the
messages and getters defined in the Tact contracts.

## Running tests

```bash
npx blueprint test
```

Or run a single contract test file:

```bash
npx blueprint test contracts/tests/agent-wallet.spec.ts
```
