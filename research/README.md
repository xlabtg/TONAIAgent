# Research Modules

This directory contains **long-term research and experimental modules** that are not part of the production platform.

These modules represent exploratory work toward future infrastructure concepts. They are **not production-ready** and should not be imported by production code.

---

## Modules

| Module | Description |
|--------|-------------|
| `agfi/` | AI-native Global Financial Infrastructure — foundational research for global capital coordination |
| `agfn/` | AI Global Finance Network — research into decentralized AI-powered financial networks |
| `gaei/` | Global AI Economic Infrastructure — economic infrastructure layer research |
| `gaamp/` | Global AI Agent Management Platform — research into large-scale agent orchestration |
| `grif/` | Global Risk Infrastructure — systemic risk monitoring and coordination research |
| `sgia/` | Systemic Global Intelligence Architecture — AI-native systemic intelligence research |
| `aifos/` | AI-native Financial Operating System — OS-level abstraction for financial AI systems |
| `acms/` | Autonomous Capital Markets Stack — vertically integrated capital markets research |

---

## Status

All modules in this directory are **research-only**:

- Not exported from `@tonaiagent/core`
- Not covered by production SLAs
- Subject to breaking changes without notice
- Intended for exploration and proof-of-concept work
